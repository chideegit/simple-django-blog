
This is a simple django blog project that has most features and fuctionalities of a modern blog site. 

User authentication: Include in this project is an app for user's authentication. So before anyone can post anything on the blog, he or she must be registered to the 
site and also be logged in.

Blog: The blog app enables an authenticated user to be able to post, update, delete and change the visibility of a blog post

The frontend was done with bootstrap and the frontend functionality was kept to a minimal, so as to enable learners focus more on understanding the backend 
architecture of the project.
